## DotMatrix


生物信息点阵图编程实现 

         生信DotMatrix 用JAVA写了个GUI 代码有待优化 另到时候打算做个网页版的

功能：
- [ ] 读入.fa文件
- [ ] 运用滑窗技术，可自定义窗长和阈值，生成对应序列点阵图
- [ ] 所有子串，并展示最长的五条坐标
